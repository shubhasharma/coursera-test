Simple Grocery List

Instructions: Add items to the list by typing in an item in the Item field, select a quantity in the Quantity field and click the Add Item button; optionally press the Enter key. Click on a row in the list in order to toggle the item's presence in the cart. Click on the Remove Items button to clear the list.
